# EPOS
[An Evolutionary Algorithm Assisted by an Ensemble of Pareto-Optimal Surrogate Models](https://ieeexplore.ieee.org/document/11589395)

- This is an open-source code of EPOS implemented in MATLAB.

- All codes are our originals and implemented to run on [PlatEMO](https://github.com/BIMK/PlatEMO/releases).


## How to run

  1. Download all the files of EPOS and [PlatEMO](https://github.com/BIMK/PlatEMO/releases).
  
  1. Add the EPOS directory in this repository to the `Algorithms/Single-objective optimization` directory.

  1. Run `platemo.m` and select EPOS. See the documents of PlatEMO for more details.
